Title
SHOP.U(Ecommerce App)
Description
SHOP.U is an Ecommerce App fully functional using the MERN(MongoDB,Express,React,Node) Stack.
Installation
Clone The Git Repository of the Project and simply run "npm install" in root directory and also in the client directory.
Usage
You can use the project as your own Ecommerce app or can use each module for your project the Project mainly focus on proving the 
functionality so you can better the UI and make your own Project.
You can use the modules,middlewares,controllers etc.
